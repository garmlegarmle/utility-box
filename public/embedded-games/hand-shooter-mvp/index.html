모드 선택

Mine Cart Duel

먼저 대전 방식을 고른다. CPU 대전은 난이도 선택 후 튜토리얼로, 다른 참가자와 대결은 튜토리얼 후 실제 온라인 매칭 플로우로 이어진다.

대결 모드를 선택해 다음 단계로 이동한다.

CPU Difficulty

난이도 선택

CPU 대전 난이도를 먼저 고른다. 선택한 난이도는 설정 패널과 게임 HUD에도 그대로 반영된다.

난이도를 누르면 바로 튜토리얼로 이동한다.

CPU Duel

튜토리얼

게임 시작을 누르면 브라우저가 카메라 사용 여부를 묻는다. 조준은 손끝으로 하고, 엄지를 접어 발사하고, 다른 손으로 슬라이드하듯 훑어 재장전한다.

Aim Pose

검지와 엄지를 편 총 모양을 유지하면 조준 상태가 된다.

Shoot

총 모양을 유지한 채 엄지를 접으면 발사된다.

Debug: Space

Reload

다른 손을 총 위로 가져가 엄지 쪽을 따라 좌우로 훑으면 재장전된다.

Debug: R

Press Start Game to grant webcam permission and begin the duel.
